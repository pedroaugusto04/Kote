# Kote Clipper - Browser Extension

A browser extension for quickly saving web pages, selections, and articles to your Kote knowledge base. Automatically converts HTML to Markdown, adds frontmatter metadata, and lets you choose the target project and tags.

## About Kote

**Kote** is a developer memory layer that automatically captures and organizes AI sessions, Git history, and development context into searchable knowledge.

For more details, visit the [GitHub Repository](https://github.com/pedroaugusto04/Kote) or the [original link](https://knowledgebase.sbs/kote).

## Installation

### Chrome Web Store (Recommended)

Install directly from the [Chrome Web Store](https://chrome.google.com/webstore/detail/kote-clipper).

## Usage

### Clipping a Page

1. Navigate to any web page you want to save
2. Click the Kote Clipper extension icon
3. The extension will automatically scan the page content
4. Review the title and edit if needed
5. Select the target project from the dropdown
6. Add tags (comma-separated) if desired
7. Click "Save Note"
8. The note will be saved and you can click "Open Note" to view it in your Kote

### Clipping a Selection

1. Highlight text or content on a web page
2. Right-click and select "Save selection to Kote"
3. The selection will be saved to your default project with a notification

### Quick Clip (Keyboard Shortcut)

1. Navigate to any web page
2. Press `Ctrl+Shift+K` (Mac: `Cmd+Shift+K`)
3. The entire page will be saved to your default project
4. A notification will confirm the save

## Keyboard Shortcuts

| Shortcut | Action | Platform |
|----------|--------|----------|
| `Ctrl+Shift+Y` | Open Kote Clipper popup | Windows/Linux |
| `MacCtrl+Shift+Y` | Open Kote Clipper popup | macOS |
| `Ctrl+Shift+K` | Quick clip full page | Windows/Linux |
| `MacCtrl+Shift+K` | Quick clip full page | macOS |
